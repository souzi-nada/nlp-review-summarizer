# NLP Review Summarizer
This project analyzes customer reviews and extracts key insights, including the most frequent pros and cons. It also generates a sentiment analysis chart for better visualization.

## Features
- Tokenizes and processes review text using NLP.
- Identifies top pros and cons.
- Generates sentiment analysis visualization.
- Outputs a concise summary paragraph suitable for e-commerce.

## How to Run
1. Clone the repository:
   git clone https://github.com/suzywilliam/nlp-review-summarizer
2. Install dependencies:
   pip install nltk pandas plotly
3. Run the script:
   python nlp_review_summary.py

## Sample Output
- Top keywords: battery, camera, performance
- Sentiment chart: Positive (35), Negative (15)
- Summary:
  "Most users praised the smartphone for its excellent battery life, high-quality camera, and smooth performance. However, common concerns included limited storage capacity, occasional overheating, and lack of fast charging support."
